SPA

SPA : ( Single Page Application )

与传统多页应用的区别

  • 传统多页应用：每次请求服务器返回的都是一个完整的单页面

  • 单页面应用：只有第一次会加载页面，以后的请求只是为了获取必要的数据，由js解析，展示在页面中。

优点与缺点

  • 优点

    1. 减少请求的体积，加快页面的响应速度，降低服务器压力。
    2. 更好的用户体验，web app 像 native app 那样流畅。

  • 缺点

    使用ajax，页面不利于seo。

    seo原则：搜索引擎在抓取页面的时候会屏蔽掉所有javascript代码，所以也就无法抓到ajax动态加载的数据。

    • 解决方案：
      1. 服务端渲染
      2. 页面静态化 ( 为爬虫准备 )
